# PossibleTech Solutions Website

## Overview
This is a simple, responsive website for **PossibleTech Solutions**, a fictional IT and software development company. The website demonstrates modern HTML and CSS techniques, including:

- **Flexbox layout** for responsive design  
- Responsive navigation and content areas  
- Styled `<article>` and `<aside>` sections  
- Custom cards for content and announcements  
- Mobile-friendly design that adjusts to different screen sizes  

The site includes two pages:

1. **index.html** – Homepage with company introduction, mission, and features.  
2. **tweets.html** – Updates page showcasing recent announcements or news.

---

## Features

- **Responsive Flexbox Layout:**  
  The website uses CSS Flexbox for both the main layout (`<article>` + `<aside>`) and the overall page (`<header>`, `<main>`, `<footer>`).  

- **Mobile Friendly:**  
  With `class="works_on_smartphone"` on the `<body>` and media queries in CSS, the layout automatically stacks for small screens.

- **Custom Styling:**  
  - Gradient backgrounds for `<article>` sections  
  - Card designs with shadows and borders  
  - Styled buttons and lists  
  - Unicode logo in the header for a unique branding element  

- **Scrollable content:**  
  Both `<article>` and `<aside>` sections allow vertical scrolling when content exceeds their container height.
